# VPS Assistant Bot 🤖

**VPS Assistant Bot** — это гибкий инструмент для мониторинга состояния ваших серверов через Telegram. Проект сочетает в себе два подхода: **безагентный мониторинг** удаленных узлов и **API-мониторинг** локального хоста.

## 🚀 Основные возможности

- 📊 **SSH-мониторинг:** Опрос удаленных серверов без установки на них дополнительного ПО (Agentless).
- 🖥 **FastAPI мониторинг:** Детальная статистика «родного» сервера (хоста), на котором запущен бот, через встроенный API-сервис.
- ➕ **Удобное управление:** Добавление и удаление серверов через пошаговый диалог.
- ⌨️ **Интерактивное меню:** Постоянная клавиатура для быстрого доступа к функциям.
- 🔒 **Безопасность:** Доступ к данным ограничен по вашему Telegram ID.
- 🐳 **Docker-native:** Полная изоляция и запуск всей системы одной командой.

## 📋 Какие данные выводит бот?

Для каждого сервера (удаленного или локального) вы получаете:
- **Статус:** Online/Offline.
- **Uptime:** Время непрерывной работы.
- **CPU:** Модель процессора и текущая нагрузка (Load Average / Usage %).
- **RAM:** Использование оперативной памяти (Занято/Всего + %).
- **Disk:** Состояние дискового пространства на корневом разделе.

## 📦 Быстрое развертывание

### 🛠 Подготовка сервера
Для запуска необходимы **Docker** и **Docker Compose**. Если они не установлены на вашей VPS, выполните официальный скрипт:
```bash
curl -fsSL https://get.docker.com -o get-docker.sh && sudo sh get-docker.sh
```

### 1. Клонирование репозитория
Склонируйте проект с GitHub и перейдите в рабочую директорию:
```bash
git clone https://github.com/NikSyrovatskij/vps-assistant-bot.git
cd vps-assistant-bot
```

### 2. Настройка переменных окружения
Создайте файл `.env` из шаблона и отредактируйте его, вставив свои данные:
```bash
cp .env.example .env
nano .env
```
**Обязательные параметры:**
- `BOT_TOKEN`: Токен вашего бота от @BotFather.
- `ADMIN_ID`: Ваш числовой ID в Telegram (можно узнать у @userinfobot).
- `API_URL`: `http://api:8000/status` (оставьте как есть для связи с FastAPI).

### 3. Запуск проекта
Запустите сборку и старт контейнеров в фоновом режиме:
```bash
docker compose up -d --build
```

## 🕹 Команды и интерфейс
- `📊 Статус серверов` — Опросить все удаленные VPS из базы данных по SSH.
- `🖥 Мой хост` — Получить мгновенную статистику текущего сервера через **FastAPI**.
- `➕ Добавить сервер` — Запустить диалог добавления нового узла (IP, User, Password).
- `📋 Список серверов` — Показать все сохраненные серверы и их ID.
- `/remove_server ID` — Удалить сервер из базы по его ID.
